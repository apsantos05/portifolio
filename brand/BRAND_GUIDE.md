# BRAND GUIDE — Murilo Manzano · Skill Educação Corporativa

**Versão:** 1.0 · Junho/2026
**Escopo:** Instagram, site pessoal, portfólio, landing pages, dashboards, sistemas, aplicações web e documentação.
**Sistema:** Navy/Azul Skill (design system oficial unificado — AElite™, Academia de IA, Skill.AI)

---

## 1. Conceito da Marca

A marca traduz visualmente a tese central de Murilo Manzano: **"do bombeiro operacional ao arquiteto de sistemas"**. A estética é a de um *blueprint executivo* — engenharia, precisão, comando. Fundo navy profundo como uma sala de controle noturna; azul periwinkle como o traço luminoso do arquiteto sobre a planta; tipografia geométrica e densa como instrumentação de precisão.

Não é uma marca "de curso". É uma marca de **sistema operacional de liderança**: sóbria, técnica, premium e tecnológica ao mesmo tempo.

### Metáfora central
> Blueprint + Cockpit. Cada peça visual deve parecer a planta de um sistema sendo instalado — grid técnico, seções numeradas, dados, glow de instrumento.

## 4. Tom de Comunicação

- **Direto e afirmativo.** Frases curtas. Verbos de comando: instalar, destravar, escalar, comandar.
- **Vocabulário proprietário:** "jornada", "sessões", "sistema", "instalação", "protocolo", "operação". **Nunca:** "curso", "aulas", "comprar", "conteúdo".
- **Bicolor verbal-visual:** em títulos, uma única palavra-chave recebe o azul periwinkle (`#84A3DA`) — a palavra que carrega a transformação. Ex.: "Liderança **100% prática**", "Do operacional ao **sistema**".
- **Números como autoridade:** métricas, ROI, percentuais sempre em display Inter Tight, grandes.
- **Inimigo nomeado:** o "Vício da Centralização" e os "3 Gargalos Invisíveis™" — a comunicação sempre contrasta o estado A (caos operacional) com o estado B (sistema instalado).

## 5. Regras de Uso

### Cores
1. Fundo padrão é **INK `#090D17`** — todo material dark-first. Versões light são exceção documentada (materiais impressos B2B).
2. **`#84A3DA` (periwinkle) é destaque, nunca fundo grande.** Máximo ~10% da área visível.
3. Botões sólidos usam **azul-deep `#3F5DAA` / `#4D6BBE`**, nunca o periwinkle puro (contraste insuficiente com texto branco).
4. Texto de corpo é `#EEF1F7` sobre navy — nunca branco puro `#FFFFFF` em blocos longos.
5. O navy corporativo B2B `#1E3A5F` é reservado para propostas/documentos institucionais Skill B2B.

### Tipografia
6. **Inter Tight** apenas para display/títulos/números. **Inter** para tudo mais. **JetBrains Mono** para código/dados técnicos.
7. Nunca usar serifadas. Playfair Display + dourado `#C9A84C` é **legado descontinuado**.

### Elementos-assinatura (usar de forma consistente)
8. Textura blueprint/grid azul sutil (`rgba(132,163,218,0.055)`, células 46px) em heros, cards de preço e CTAs.
9. Títulos bicolores (branco + 1 palavra azul).
10. Botões pill com ícone circular de seta.
11. Seções numeradas `01 — 06` com eyebrow em uppercase tracking largo.
12. Glow radial azul (`rgba(63,93,170,0.26)`) posicionado fora do centro.
13. Bloco de preço com ancoragem: valor de referência riscado → condição especial → crédito → investimento final em destaque sobre azul translúcido.

## 6. Boas Práticas

- **Menos elementos, mais hierarquia.** Cada tela/slide tem UM ponto focal.
- **Respiro:** mínimo 32px de padding lateral em mobile, 80–120px de espaçamento vertical entre seções em web.
- **Grid antes de tudo:** todo layout nasce do grid de 8px e do container de 1080–1200px.
- **Animações com propósito:** scroll-reveal suave, strikethrough animado em ancoragem de preço, glow estático. Nada de bounce, flash ou confete.
- **Acessibilidade:** contraste mínimo AA (texto `#EEF1F7` sobre `#090D17` = ~15:1 ✓; periwinkle sobre navy só em textos ≥14px bold ou elementos decorativos).
- **Consistência entre canais:** o mesmo token que colore um botão no site colore o CTA de um carrossel do Instagram.

## 7. Erros que NUNCA devem acontecer

| ❌ Erro | Por quê |
|---|---|
| Usar dourado `#C9A84C` ou Playfair Display em material novo | Sistema legado descontinuado (mai/2026) |
| Preto puro `#000000` como fundo | Mata a profundidade do navy; usar `#090D17` |
| Periwinkle `#84A3DA` como fundo de botão com texto branco | Contraste insuficiente; usar `#3F5DAA`/`#4D6BBE` |
| Mais de uma palavra azul por título | Dilui o efeito bicolor |
| Gradientes multicoloridos, neon verde/roxo | Fora da paleta; quebra a identidade |
| Emojis em títulos de materiais premium (AElite™, propostas) | Quebra o tom executivo |
| Vocabulário "curso/aula/comprar" | Contradiz o posicionamento de "sistema/instalação" |
| Sombras pretas duras (`box-shadow` sem cor da marca) | Sombras devem carregar navy/azul translúcido |
| Border-radius inconsistente (misturar 4px, 10px, 24px na mesma peça) | Usar a escala oficial (ver COMPONENTS.md) |
| Fotos com filtros quentes/alaranjados | Tratamento de imagem deve ser frio/azulado |
